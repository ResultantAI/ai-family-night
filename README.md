# AI Family Night

**Turn screen time into together time.**

A weekly 15-minute AI challenge for parents and kids to create something fun together.

🌐 **Live site:** [aifamilynight.com](https://aifamilynight.com)

---

## What is this?

AI Family Night delivers one creative challenge per week. Parents and kids do it together—no tech skills required. Every challenge produces something tangible: a comic, a story, artwork, a game design.

**The point isn't to teach kids about AI. It's to stop feeling like a stranger in their digital world.**

---

## Example Challenges

- **Roblox Mirror** — Create a silly "Black Mirror for kids" comic starring your child
- **Dream Treehouse** — Design your ultimate treehouse with AI-generated images  
- **Superhero Origin Story** — Turn your kid into a hero based on their real personality
- **Family Theme Song** — Write a ridiculous song about your family

---

## Tech Stack

- Static HTML/CSS/JS (no framework)
- GitHub Pages hosting
- ConvertKit for email capture
- Plausible Analytics (privacy-first)

---

## Local Development

```bash
# Clone the repo
git clone https://github.com/[username]/ai-family-night.git

# Open in browser
open index.html
```

No build step. Just HTML.
